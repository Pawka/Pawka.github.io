TIL: Spotify media keys on Linux

2020-09-07
til linux

:til: :linux:

It seems Chrome is capturing media keys handling and Spotify can not be controlled with the keys. This can be solved by visiting chrome://flags/#hardware-media-key-handling and disabling “Hardware Media Key Handling” option.
